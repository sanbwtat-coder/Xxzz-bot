Fadi Protection Maker

الملفات:
- maker.py
- config.py
- protection_template.py
- requirements.txt

التشغيل على Railway:
1. افتح config.py وضع توكن بوت الصانع في MAKER_TOKEN.
2. ارفع الملفات.
3. Start command: python maker.py

المميزات:
- صنع بوت حماية فقط.
- مجاني: اشتراك إجباري بقناتك @fadifva.
- مدفوع: صاحب البوت يغير قناة الاشتراك.
- أكواد تفعيل: شهر، شهرين، 3 أشهر، 6 أشهر، سنة.
- لوحة مطور فيها عدد المستخدمين والبوتات والمدفوع والمجاني.
- إشعار للمطور عند صنع بوت جديد.
- أزرار ملونة style primary و SOURCE FADI باللون الأحمر style danger.
